# **App Name**: ChatterBox

## Core Features:

- Nickname Input: Allow users to enter a nickname to identify themselves in the chat.
- Real-time Chat: Enable real-time messaging between users.
- Message Saving: Persist messages in the Firestore database so they are available even after the user leaves and returns.
- User Identification: Display the user's nickname next to their messages.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calming and friendly feel.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Dusty rose (#D8BFD8) for subtle highlights and user interaction.
- Body and headline font: 'PT Sans' for a modern and clean appearance.
- Use simple, outlined icons for user actions and system messages.
- A clean and intuitive layout with clear message separation.
- Subtle animations for new messages and user interactions.
